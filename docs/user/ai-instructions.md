# AI Instructions

## What It Does

Architecture Studio can now generate AI guidance and an `AGENTS.md` file from the selected or detected project profile.

## What Gets Included

The generated guidance includes:

- architecture rules
- coding standards
- devops rules
- compliance requirements

## Why It Matters

This gives coding agents a project-specific rule set instead of a generic prompt. The generated guidance is based on the same standards, compliance, and architecture outputs already used elsewhere in the extension.

## Current Outputs

The current generator produces:

- `AGENTS.md`
- `docs/ai-instructions.md`

## Command

Use `Architecture Studio: Generate AI Instructions` to generate the AI guidance artifacts.
